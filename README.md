## Pasos para ingresar correctamente a la página web creada

1. Ingresar al link del proyecto compartido.

2. Al abrir el enlace, Lovable puede pedir iniciar sesión. Se puede ingresar con Google, GitHub, Apple o correo electrónico.

3. Aceptar las cookies de la página para continuar.

4. Aceptar los permisos de acceso solicitados por Lovable.

5. Luego, completar la configuración inicial de la aplicación. La plataforma puede preguntar:
   - Nombre del usuario.
   - Rol que mejor describe lo que hace.
   - Cantidad de personas que trabajan en la empresa.
   - Preferencia de estilo: modo claro u oscuro.

6. Una vez completadas esas preguntas, cerrar esa pestaña y volver a ingresar al link original del proyecto.

7. Al volver a entrar, aparecerá una invitación al proyecto. Presionar *Accept invitation*.

8. Finalmente, se podrá visualizar y utilizar la página web creada.
